## Learning Objectives

By the end of this chapter, you will understand:

* What **props** are in React
* How to pass data from a parent component to a child component
* How to use JavaScript inside JSX
* How to render lists dynamically using `.map()`
* Why React requires a `key` prop when rendering lists

## 2. What Are Props?

**Props (Properties)** are values passed from a parent component to a child component.

Think of props like function arguments.

# Chapter Summary

React Props allow components to receive data from their parent components. Props make components reusable and dynamic. JSX uses curly braces `{}` to embed JavaScript expressions. React commonly uses `.map()` to render lists, and each list item should have a unique `key` so React can efficiently update the UI.
