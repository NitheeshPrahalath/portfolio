# Need for Data Structures

    Processing speed due to complex and high amount of data

    Traversing across a data item in a store of considerable size will take a lot of time.

    Multiple requests from thousands of clients at the same time simultaneously will cause conflicts and inconsistency of the data processed.

# Advantages of DS

    Efficiency - If the choice of a data structure for implementing a particular ADT is proper, it makes program very effiicient in terms of time and space.
